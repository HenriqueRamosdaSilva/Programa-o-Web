JavaScript (JS) é uma linguagem de programação de alto nível, interpretada e dinâmica, essencial para tornar sites interativos e funcionais. Criada para navegadores, hoje ela é usada no front-end e back-end (via Node.js). Ela adiciona comportamento, como animações, manipulação de conteúdo (HTML/CSS) e validação de formulários.
